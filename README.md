to download all the node modules and dependencies for both the backend and admin panel. 

```bash
npm run build
```
it would run on the root package json file that also directs to the backend and admin panel package json files to install the dependencies and build the admin panel.

to start the backend server, you can run the following command:

```bash 
npm run start
```
also configures from the root package json file to start the backend server.
