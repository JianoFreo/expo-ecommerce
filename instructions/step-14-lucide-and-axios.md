install lucide for icons and axios for data fetching


Lucide is a library of open-source icons that can be used in web applications. It provides a wide range of icons that are customizable and easy to use. With Lucide, you can easily add icons to your project without having to create them from scratch.

in this code, we are installing the `lucide-react` package, which is a React wrapper for the Lucide icon library. This allows us to use Lucide icons in our React application. The `npm install lucide-react` command will download and install the package, making it available for use in our project.
```bash
cd admin
```
```bash
npm install lucide-react
```

axios is used for data fetching, so let's install it as well:
axios is a promise-based HTTP client for the browser and Node.js. It provides an easy-to-use API for making HTTP requests and handling responses. With axios, you can perform various types of requests such as GET, POST, PUT, DELETE, and more. It also supports features like interceptors, request cancellation, and automatic JSON data transformation.

in this code, axios is used to fetch data from the API and handle the response. The `fetchData` function makes a GET request to the specified URL and logs the response data to the console. If there is an error during the request, it catches the error and logs it as well.
```bash
npm install axios
```
